# PBI-001 Shariah Review Questions

## Meeting purpose
Validate the proposed MVP PLS model before implementation PBIs are created.

## Proposed baseline for review
- Contract family: restricted mudarabah
- Scope: single venture linked to one procurement reference
- Goal: minimal Shariah-valid MVP within 12-week delivery constraint

## Questions

1. Is the proposed “single-venture restricted mudarabah” acceptable as an MVP framing?
2. Can the procurement reference (PO / awarded tender / job) be used as the restriction scope?
3. How should “realized profit” be defined in this context?
4. Which cost categories may be deducted before profit distribution?
5. Is manual approval required before every distribution event?
6. What evidence is sufficient to determine negligence / misconduct / breach?
7. Are any security or collateral arrangements permissible without becoming an impermissible guarantee?
8. Are there any mandatory disclosure or documentation fields missing from the draft schema?
9. Is it acceptable to defer musharakah support to post-MVP?
10. What wording should appear in user-facing contract summaries to avoid implying guaranteed return?

## Required outputs from meeting
- decision / recommendation
- conditions or caveats
- reviewer name and role
- date
- approval status:
  - approved
  - approved with conditions
  - requires revision
  - rejected
